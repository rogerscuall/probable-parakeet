# Shared Resources

This directory contains resources that are shared across all design types.

## Directory Structure

```
shared/
├── images/      # Common images (logos, diagrams, icons)
├── markdown/    # Common documentation snippets
├── templates/   # Common code/config templates
└── schemas/     # JSON schemas for content validation
```

## Usage

### Images
Place common images here that can be referenced from any design type's markdown:
- Company logos
- Generic architecture diagrams
- Icon sets
- Common UI elements

### Markdown
Reusable markdown snippets:
- Common headers/footers
- Legal disclaimers
- Standard sections (prerequisites, glossary)

### Templates
Shared template files:
- Common configuration patterns
- Base template structures

### Schemas
JSON schemas for validating:
- Label configuration files
- Manifest files
- Content structure
